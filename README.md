# TheKeenan
A forward model to estimate stellar parameters (e.g., Teff, logg, [Fe/H] and chemical abundances) using Support Vector Regression (SVR), which in essential is a non-parametric method.


# Installation
`pip install TheKeenan`


# Requirements

- numpy
- scipy
- matplotlib
- astropy
- sklearn
- joblib
- pandas
- emcee
